# HPC Web 管理平台

一个功能完整的高性能计算集群 Web 管理平台，使用 Vue 3 + TypeScript + Vite 构建。

## 功能模块

### 1. 📊 仪表盘
- 平台概览：节点状态、CPU/GPU 数量统计
- 资源使用率：实时监控 CPU、GPU、内存使用情况
- 作业统计：运行中、等待中、已完成作业数量
- 节点状态表：详细展示每个节点的资源使用情况

### 2. 📋 作业管理
- 作业信息：查看个人作业状态和统计
- 提交作业：通过表单提交新的计算任务
- 作业模板：预设常用作业配置模板
- 所有作业：查看集群所有用户的作业列表

### 3. 💻 Web Shell
- 在线终端：浏览器内访问集群命令行
- 命令执行：支持 Slurm 命令（squeue、sinfo 等）
- 实时交互：即时命令响应和输出

### 4. 🖥️ 远程桌面
- VNC 连接：远程访问计算节点图形界面
- 节点选择：支持连接不同计算节点
- 工具栏：全屏、截图、剪贴板等功能

### 5. 📁 文件管理
- 文件浏览：可视化浏览集群文件系统
- 文件操作：上传、下载、重命名、删除
- 快捷访问：主目录、临时目录、数据目录
- 文件搜索：快速定位文件

## 快速开始

### 安装依赖
```bash
cd hpcweb
npm install
```

### 启动开发服务器
```bash
npm run dev
```

访问 http://localhost:3000

### 构建生产版本
```bash
npm run build
```

## 技术栈

- Vue 3 - 渐进式 JavaScript 框架
- TypeScript - 类型安全
- Vite - 快速构建工具
- 原生 CSS - 无需额外 UI 库

## 后续集成

当前使用模拟数据，需要集成实际的 Slurm API：

1. 替换 `App.vue` 中的 `loadJobs()` 函数，调用真实 API
2. 替换 `JobSubmit.vue` 中的 `submitJob()` 函数
3. 添加任务取消、日志查看等功能
